Testar localmente

Para testar esta página localmente (e o projeto):

git clone git@github.com:Maxicad/Valida-IFC-By-Codex.git
cd Valida-IFC-By-Codex
npm install
npm start
# Abra http://localhost:3000/docs/visual-v2/

(O npm install pode não ser necessário se o projeto for estático.)
